# PANGEOS Clothing Industries 🌱👕

Official website for **PANGEOS Clothing Industries**, a premium apparel brand focused on **sustainable, cotton-based fashion**.  
The platform highlights the brand’s vision, collections, and commitment to eco-friendly clothing.

> **Scrap Polyester. Choose Cotton. Stay Natural.**

---

## 🌍 Project Overview

This project is a modern, SEO-optimized web application built to represent **PANGEOS Clothing Industries** online.  
It is designed for performance, accessibility, and scalability, ensuring a smooth experience across all devices.

---

## 🚀 Tech Stack

- **HTML5**
- **React + TypeScript**
- **Vite**
- **CSS / Tailwind CSS**
- **SEO & Social Media Meta Tags**
- **Responsive Design**

---

## 🧠 Key Features

- ✅ SEO-friendly meta tags (Google, Facebook, Twitter)
- ✅ Mobile-first responsive layout
- ✅ Fast build & hot reload using Vite
- ✅ Scalable component-based architecture
- ✅ Clean, production-ready structure
- ✅ Brand-focused content & messaging

---

## 🧩 SEO & Metadata

The project includes:

- **Primary meta tags** for search engines
- **Open Graph tags** for Facebook & LinkedIn sharing
- **Twitter Card metadata**
- **Canonical URL** for duplicate-content prevention

This helps improve:

- 🔍 Search engine visibility  
- 📈 Click-through rate  
- 🌐 Social media preview quality
